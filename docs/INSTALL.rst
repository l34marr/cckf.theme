cckf.theme Installation
-----------------------

To install cckf.theme using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``cckf.theme`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        cckf.theme
       
* Re-run buildout, e.g. with:

    $ ./bin/buildout
